# Definition of Success (Критерії успіху MVP)

MVP вважається успішним, якщо виконані такі умови:

## 1. Критерії успіху (Observation-based)
1. Не менше 60% тестових користувачів (3 з 5) можуть самостійно знищити ворога без пояснень.
2. Час виконання головного сценарію (від запуску до знищення ворога) не перевищує 120 секунд.
3. Кількість критичних технічних збоїв дорівнює 0 під час 10 послідовних запусків.
4. Мінімум 3 користувачі підтвердили, що керування зрозуміле без інструкцій.
5. Отримано щонайменше 3 текстових відгуки про зручність і відчуття стрільби.

## 2. Метрики
- Час проходження головного сценарію (секунди)
- Відсоток користувачів, що завершили сценарій без підказок
- Кількість критичних багів та крашів (0 допустимо)
- Суб’єктивна оцінка користувача (цікаво / нейтрально / нецікаво)
- Кількість запитань від користувачів під час тестування
- Стабільність FPS (без різких просідань)

## 3. Невизначеності та ризики
- Чи буде керування зручним на різних пристроях.
- Чи буде базова механіка стрільби достатньо цікавою.
- Чи не занадто швидко з’явиться нудьга.
- Способи зменшення: мінімальний функціонал, тестування на 5 користувачах, спрощення механік до однієї зброї.
